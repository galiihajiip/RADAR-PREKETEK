# Design System

| Token | Hex | Usage |
|---|---|---|
| Primary Navy | `#0B1F3A` | identity, primary actions |
| Command Blue | `#0F4C81` | hover and dashboard accents |
| Radar Cyan | `#00A8CC` | focus, signal, logo |
| Disaster Red | `#D62828` | destroyed/critical |
| Warning Orange | `#F77F00` | major damage |
| Safe Green | `#2A9D8F` | no damage/success |
| Alert Yellow | `#F4D35E` | minor damage |

Typography uses Inter, bold compact headings, 16px body baseline, and tabular numerals for metrics. Components: `RadarLogo`, `SeverityBadge`, `OnlineStatusBadge`, `RoleBadge`, `MetricCard`, `SectionHeader`, `EmptyState`, `LoadingState`, and `ErrorState`.

Do: keep emergency UI calm, high contrast, and scannable. Do not: use generic purple gradients, decorative clutter, or severity colors inconsistently.
